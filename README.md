# Кинотеатр job4j_cinema

## О проекте

Приложение "Кинотеатр".
В этом учебном приложении разработан сервис по покупке билетов в кинотеатр.
Сервис предназначен для одного кинотеатра.

## Стек технологий
- Java 17
- Spring Boot 2.7.6
- Maven 3.8
- Git
- PostgreSQL 14
- Thymeleaf 
- Bootstrap 5.2.3

## Требования к окружению
- Java 17
- Maven 3.8
- PostgreSQL 14
